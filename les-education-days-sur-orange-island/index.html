Les Education days sur Orange Island

Aujourd’hui et demain aura lieu sur Orange Island les Educations days : 2 jours de conférences sur le thème de l’éducation dans les mondes virtuels. Encore une fois, et c’est le choix d’Orange, les horaires sont plutôt orientés USA mais les intervenants sont de qualité, je pense que cela vaut le coup de rester éveillé.

<p>
  Voici le programme avec les <span style="font-weight: bold;">heures françaises</span> :
</p>

<p>
  <b><u>Mardi 28 Avril 2009 : </u></b>
</p>

19.00 : Open discussion with Dusan Writer: ‘Education in virtual worlds, where do we stand?’
Intervenant : Dusan Writer (Metanomics, Remedy Communications)

20.30 : Sleep is for babies: Friendly scheduling, cohesion across international date/time lines
Intervenants : Viv Trafalgar (Aether Salon, Armada-Breakaway sim), DreddPirateBob Streeter (Preston College, Armada-Breakaway sim)

22.00 OpenHabitat, using multi-user virtual environments in higher education
Intervenants : Wawrick Donat (Oxford University), Cubist Scarborough (Leeds Metropolitan University)

23.00: MUVEnation, training teachers for multi-user virtual environments
Intervenants : Paz Lorenz (MENON Network EEIG, Belgium) & StevenW Bohm (King’s College, London)
