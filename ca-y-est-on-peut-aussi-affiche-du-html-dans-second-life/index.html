Ca y est on peut aussi affiché du HTML dans Second Life


La nouvelle est tombé le 6 Mars sur le blog de LL cela ne concerne pour le moment que la version release candidate mais c’est enfin possible sur SL.
Je pense qu’ils n’avaient pas trop le choix en fait avec tout le bruit qu’a fait la sortie de RealXtend.
Il est donc possible d’afficher du html ( une page web ) sur des objets. Par contre juste du HTML c’est a dire pas de refresh des pages, et ma connaissance pas de VNC non plus !
Sont -ils toujours en retard sur Opensim? Moi je pense que oui.
Dur dur de battre le monde de l’opensource en terme de réactivité.

En parlant d’opensource, la version Linux de cette release candidate ne veut tout simplement pas ce lancer, crash dès le “precaching”. Les linden doivent vraiment avoir une dent contre les logiciels libres.

Autre petite révoltution dans le monde de Second Life : les vidéos peuvent être maintenant dans un autre format que ce bon vieux quicktime, à savoir MPEG.
Enfin un format à peu près universel – même si propriétaire -, cela devrait arranger beaucoup de monde !
