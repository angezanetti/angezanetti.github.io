Les mondes 3d directement dans votre browser c’est – presque – fait !

La nouvelle me vient d’un tweet reçu ce matin et poster par Hugo ( ” Hugobiwan via seekeur : sl directly in you browser http://ajaxlife.net/ ” )

J’ai donc bien évidemment sauté sur le lien pour tester un peu tout ça. La page d’accueil ressemble fortement a celle du client officiel si ce n’est le : “Unofficial viewer AjaxLife “.



La bonne nouvelle c’est le petit menu déroulant permettant de ce connecter directement aussi bien au monde de Linden Labs qu’à des plates formes libres.
