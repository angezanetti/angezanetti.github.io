Les réseaux sociaux pour les métaverses débarquent sur le web!

Deux sites ont attirés mon attention dans ma blogroll, myRl & Teleporteur.com.
Ces deux sites proposent des services bien connus des internautes adepte du web 2.0 : garder le contact avec ces “amis”, partager, voter etc….
Les deux sites sont en cours de construction.
myRL est clairement en beta, il faut un code pour y accéder. J’ai fait la demande hier, rien de neuf pour l’instant ( Hugo si tu as des précisions à apporter il me semble que tu as un compte myRL).
Teleporteur est pour l’instant un annuaire des sites traitant de SL , mais aussi des avatars voulant mettre en avant leurs compétences. Une fonctionnalité sympa devrait arrivée bientôt, le partage de LandMark. Sympa l’adaptation d’un Netvibes pour SecondLife.

Ces sites auront -ils autant de succès que sur le web 2d? A suivre…
