Le SDK pour le Kinect de Microsoft est sorti !

Tous ceux qui ont bidouillé le Kinect l’attendais avec impatience je pense, le SDK officiel de Microsoft est disponible depuis quelques heures.

Vous pouvez le télécharger ici : http://research.microsoft.com/…

Bon première remarque il n’est disponible que sous Windows 7. Autrement dit les possesseurs de Mac, Linux ou autre Windows Vista peuvent se gratter :)

Deuxième remarque c’est de la Beta, gratuite pour le moment mais une version commerciale devrait voir le jour dans les prochains mois (source).

D’après la documentation que j’ai téléchargée avec ce SDK il est possible de gérer la capture du squelette, la caméra et les distance mais aussi l’audio processing avec une intégration de l’API pour la reconnaissance vocale de Microsoft.

Dommage que tout cela sorte en version propriétaire, d’autant que je ne suis pas sur que la communauté des développeurs qui bossaient avec les drivers “hackés” feront le pas vers ce SDK – surtout si c’est payant et contraint à l’IDE Visual Studio.

En tout cas je ne sais pas vous mais moi j’ai bien envie de le tester ce SDK, reste à me mettre au C# !
